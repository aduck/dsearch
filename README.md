# dsearch
  > 使用命令行抓取豆瓣电影院数据

### install
  ```bash
  $ npm install dsearch -g
  ```

### usage
  ```bash
  $ dsearch -V # 0.1.0
  $ dsearch --help # get help
  $ dsearch nowplay # GET nowplaying movies
  $ dsearch top10 [-t 全国] # playing movies top10
  $ dsearch willplay # GET will play movies
  ```
